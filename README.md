# data-structure-practice
数据结构与算法练习仓库，记录C++数据结构学习、课程作业、刷题的完整代码实现。

## 仓库目录结构
| 模块 | 内容 |
| :--- | :--- |
| 01-线性表 | 顺序表、单链表、双链表、循环链表 
| 02-栈与队列 | 顺序栈、链栈、队列、循环队列 
| 03-串 | 字符串匹配、KMP算法 
| 04-数组与广义表 | 稀疏矩阵、多维数组 
| 05-树与二叉树 | 二叉树、平衡树、堆、哈夫曼树 
| 06-图 | 图的遍历、最短路径、最小生成树 
| 07-查找 | 二分查找、哈希表、B树 
| 08-排序 | 冒泡、快排、归并、堆排等 

## 学习目标
- 夯实数据结构基础，为考研、算法竞赛、后端开发打牢根基
- 完成课程作业、刷题练习，形成完整的技术作品集
- 规范代码结构，养成良好的工程化开发习惯
